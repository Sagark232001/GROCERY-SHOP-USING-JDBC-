
Grocery Shop Management System Overview
   This project is a simple Grocery Shop Management System implemented in Java using JDBC. The application allows users to manage products within a grocery shop. 

Technologies Used
    * Java
    * MVC 
    * JDBC (Java Database Connectivity)
    * POSTGRESQL  (as the database)

Project Structure

    |-- src
    |   |-- controller
    |   |   |-- Controller.java
    |
    |   |-- model
    |   |   |-- Product.java
    |
    |   |-- view
    |   |   |-- View.java

How to Run
    * Clone the repository.
    * Open the project in your favourite Java IDE.
    * Compile and run the View.java file.

Dependencies
    * POSTGRESQL  Connector/J (JDBC Driver)
